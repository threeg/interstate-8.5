## Through-line

The **highway / interstate** motif is consistent across the site's history — **v1 put highway scenes in
the header** (in album-themed colours), **v2 put a highway image in the background** — and the name
itself (I-8) is a highway reference. Keep it as a design element in v5; the layout, though, should
improve on v2, not copy it.
